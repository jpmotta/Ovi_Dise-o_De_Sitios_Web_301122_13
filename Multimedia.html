Qué es Html 5

HTML5 es un lenguaje markup (de hecho, las siglas de HTML significan Hyper Text Markup Language).
Es el lenguaje de marcado estándar para crear páginas web.
La estructura se realiza a través de marcado.
Los elementos HTML son representados por etiquetas.
